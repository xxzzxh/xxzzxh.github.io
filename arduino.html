Row-columm Scanning to control an 8x8 LED Matrix

所需硬件

  • Arduino开发板 X 1
  • 200欧姆电阻 X 2
  • 连接线 X n
  • 面包板 X 1
  • 开关 X 2
  • 1588LED8*8点阵 X 1

示意图

电路图

矩阵的16个引脚连接到Arduino板的16个引脚，六个模拟引脚用作数字输入，通过200欧姆电阻将开关与arduino开发板相连，连接到引脚12和13的2个开关控制1588LED8*8点阵中点亮的LED进行“向上”/“向左”运动。
